# Anti-Gravity Project Blueprint: Stitch Café Digital Menu

## 1. Project Overview
**Name:** Stitch Café Menu (Table 12)
**Theme:** "Elite" / "Anti-Gravity" - characterized by glassmorphism, floating UI elements (3D transforms, shadows), and high-end aesthetic.
**Tech Stack:** HTML5, Tailwind CSS (via CDN), Vanilla JavaScript.
**State Management:** Client-side `localStorage` for cross-page cart synchronization.

## 2. Architecture & Design System

### Visual Style ("Elite" Theme)
- **Glassmorphism:** Extensive use of `.glass-panel`, `.glass-search` with `backdrop-blur` and semi-transparent backgrounds.
- **Floating Elements:** Decor elements (e.g., `.green-float`, `.chili-float`, `.crumb-particle`) positioned absolutely with transforms to create depth.
- **Typography:** `Playfair Display` (Serif) for headings, `Plus Jakarta Sans` or `DM Sans` (Sans) for body text.
- **Color Palettes:** Category-specific palettes (e.g., Caramel/Beige for Bakery, Orange/Yellow for Brunch, Teal/Cyan for Cold Drinks).

### Data & Logic
- **Cart System:**
  - **Storage:** `localStorage` keys:
    - `stitch_cart`: Array of items `{ id, name, price, qty }`.
    - `stitch_cart_count`: Total item count.
    - `stitch_cart_total`: Total price.
  - **Synchronization:** `getStoredCart()` and `updateBar()` functions present on all product pages to ensure the floating cart bar reflects the global state.
  - **IDs:** Product IDs are generated from names (e.g., `bakery-chocolate-babka`).

### Image Strategy
- **Source:** Unsplash (high quality, royalty-free).
- **Implementation:** **Pinned IDs** (e.g., `source.unsplash.com/ID/size`) are used instead of query-based URLs (`source.unsplash.com/?query`) to ensure **deterministic visuals** (images don't change on refresh) and prevent ORB/loading issues.
